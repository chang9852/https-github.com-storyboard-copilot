## 修复

- 修复 reqwest 缺少 `rustls-tls` 特性导致所有 AI 供应商 HTTPS 请求直接失败的问题（根本原因：请求从未发出，积分不扣除、网站无请求记录）
- 修复 AI provider HTTP 客户端无超时配置，请求可能永久挂起导致 UI 卡在"正在提交任务..."（新增 15s 连接超时 + 300s 总超时）
- 修复应用启动时 Zustand persist 未恢复完成就调用生图接口，导致传入空 API key 触发 401 的问题
- 修复 hydration 等待无超时兜底，若 localStorage 异常则前端永远卡在"正在提交任务..."（新增 5s 超时）
