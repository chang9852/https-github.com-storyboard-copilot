## 新增

- AI 生成请求全面通过 Tauri 安全网关，彻底消除浏览器 fetch 直连 API，API Key 不再暴露在前端网络层
- 持久化层由 localStorage 迁移为 SQLite（WAL 模式），支持事务、防抖写盘与图片引用回收
- 双通道持久化：整量项目快照（260ms 防抖 + 空闲调度）与视口轻量更新（280ms 防抖），拖拽不再卡顿
- 新增 `generateImage.ts` 工具函数，统一提交+轮询模式，供后续组件迁移复用
- Rust 后端新增 `project_image_refs` 表与孤立图片自动清理机制

## 优化

- `tauriAiGateway` 升级，KIE/fal 模型自动将参考图转为 data URL，其他模型自动持久化到本地再传入
- `useSettingsDraft` 草稿模式验证—修改即时生效的写入逻辑已被 draft 机制替代
- 版本号三地同步（package.json / Cargo.toml / tauri.conf.json），避免历史版本偏差

## 修复

- 删除无效代码文件 `sqlitePersistence.ts`（命令未注册，结构体不匹配）、`projectStoreEnhanced.ts`（死代码）、`imageReferencePool.ts`（旧哈希方案被内联索引替代）
