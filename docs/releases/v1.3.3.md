## 优化

- 重构画布主流程，抽离项目同步、快捷键和节点创建菜单，降低画布组件复杂度。
- 统一节点注册表中的持久化元数据，减少新增节点时的重复映射配置。
- 拆分项目持久化映射与写入队列，让项目 Store 更聚焦状态管理。
- 拆分标注工具编辑器，将工具栏、舞台渲染、编辑状态和纯工具函数分离。
- 抽离 Tauri 侧公共 SQLite 打开逻辑，统一数据库路径、WAL 和 busy timeout 配置。
- 增加浏览器预览降级处理，Vite 预览可正常打开项目管理页。

## 修复

- 修复浏览器预览中标题栏直接调用 Tauri Window API 导致错误边界的问题。
- 修复设置持久化 hydration 阶段自引用 store 造成控制台错误的问题。
