## 新增

- 完整设计系统迁移：CSS 变量 token 体系，支持明暗主题、暖冷色调、圆角预设、强调色自定义
- 设置对话框：密钥管理、外观（主题/色调/圆角/强调色）、价格显示、实验性开关、关于页面
- i18n 重构：TitleBar、LeftSidebar、ProjectManager 等组件全面国际化
- 语言切换按钮（TitleBar 右侧）
- API Key 配置状态指示（侧边栏）

## 优化

- TitleBar 使用 lucide-react 图标替代内联 SVG，统一设计 token
- LeftSidebar 使用 CSS 变量替代硬编码颜色，增加项目重命名功能
- ProjectManager 卡片圆角使用 `--ui-radius-lg` 变量
- App.tsx 劝始化时应用 data-ui-radius、data-theme-tone、data-platform、--accent 等运行时主题属性
